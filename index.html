Acerca de este ejercicio

  • La fuente se obtuvo desde datos abiertos del país Colombia, y contiene datos del Ministerio de Educación (url: https://www.datos.gov.co/Educaci-n/MEN_ESTADISTICAS_EN_EDUCACION_EN_PREESCOLAR-B-SICA/sras-4t5p), una vez seleccionado este conjunto en lugar de otros , el conjunto seleccionado se descarga en formato csv que se adjunta, el cual nos permite entender la problemática de manera que lográramos obtener respuestas.
  • Para dibujar el mapa de Colombia, nos apoyamos en un proyecto github (repositorio público https://github.com/marisbotero/Colombia-map-with-D3) que emplea GeoJSON y D3.js para codificar cada departamento pues, para graficar con geojson desde cero cuando no existe una base, la labor es muy extensa (se requiere indicar coordenadas por cada elemento a delimitar en el mapa).
  • Plantilla bootstrap usada: https://startbootstrap.com/template/scrolling-nav
